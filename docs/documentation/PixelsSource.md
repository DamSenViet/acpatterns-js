# PixelsSource

A `class` for overloaded and remapped pixel grid acessors.

## Instance Accessors

### width

The width of the pixel grid.

* `get`
  + Type: `number`

### height

The height of the pixel grid.

* `get`
  + Type: `number`

### reactive

The reactive map of the pixels.

* `get`
  + Type: `FixedLengthArray<FixedLengthArray<paletteIndex>>`

### unreactive

* `get`
  + Type: `FixedLengthArray<FixedLengthArray<paletteIndex>>`
